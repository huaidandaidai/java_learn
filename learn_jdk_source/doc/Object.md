#Object 类详解
Object类位于java.lang包下，全类名为java.lang.Object
## Object类的作用
Object类为Java语言中的万类之祖，所有的类都继承自Object类，并且实现Object类的方法,详细解释见类注释。
Object类包含一个静态代码块，12个方法(1个private、1个protected、10个public)
